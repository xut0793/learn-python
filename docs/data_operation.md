# 数据运算

Python 数据的运算，主要包括以下几类：算术运算、关系运算、逻辑运算、位运算、成员运算、身份运算，以及运算符的优先级。

## 运算符和操作数

举个简单的例子 `4 +5 = 9` 。 其中，`4` 和 `5` 被称为操作数，`+` 称为运算符。

## 算术运算

算术运算符用于执行基本的数学运算，返回计算结果。

| 运算符 | 简单描述                         | 例子         |
| ------ | -------------------------------- | ------------ |
| +      | 加法，用于两个数相加             | 4 + 5 = 9    |
| -      | 减法，用于两个数相减             | 8 - 3 = 5    |
| \*     | 乘法，用于两个数相乘             | 3 \* 4 = 12  |
| /      | 除法，用于两个数相除             | 10 / 2 = 5.0 |
| //     | 整除，返回商的整数部分(向下取整) | 10 // 3 = 3  |
| %      | 取模，返回除法的余数             | 10 % 3 = 1   |
| \*\*   | 幂运算，返回 x 的 y 次幂         | 2 \*\* 3 = 8 |

通常情况下，整数运算结果还是整数，如果整数与浮点数运算结果是浮点数。但是需要注意的特殊点是整数的除法运算，返回的是一个浮点数。

> 向下取整除，就是地板除 floor division，即 向下取整
>
> 向上取整除，就是天花板除，ceil division，即 向上取整
>
> 向下取整是朝着数值变小的方向取整，而向上取整是朝着数值变大的方向取整

## 关系运算

关系运算符用于比较两个数的大小，返回 True 或 False。

| 运算符 | 简单描述                                   | 例子   |
| ------ | ------------------------------------------ | ------ |
| ==     | 等于，判断两个数是否相等                   | 4 == 4 |
| !=     | 不等于，判断两个数是否不相等               | 4 != 5 |
| >      | 大于，判断第一个数是否大于第二个数         | 4 > 3  |
| <      | 小于，判断第一个数是否小于第二个数         | 3 < 4  |
| >=     | 大于等于，判断第一个数是否大于等于第二个数 | 4 >= 4 |
| <=     | 小于等于，判断第一个数是否小于等于第二个数 | 3 <= 4 |

## 逻辑运算

逻辑运算符用于连接多个条件，返回 True 或 False。

| 运算符 | 简单描述                                        | 例子            |
| ------ | ----------------------------------------------- | --------------- |
| and    | 逻辑与，两个数都为 True 时，返回 True           | 4 > 3 and 5 > 2 |
| or     | 逻辑或，两个数有一个为 True 时，返回 True       | 4 > 3 or 5 > 2  |
| not    | 逻辑非，将 True 转换为 False，False 转换为 True | not 4 > 3       |

## 成员运算

成员资格运算符用于测试序列是否在对象中出现，返回 True 或 False。

| 运算符 | 简单描述                                 | 例子                     |
| ------ | ---------------------------------------- | ------------------------ |
| in     | 成员运算，判断一个元素是否在一个序列中   | 4 in [1, 2, 3, 4, 5]     |
| not in | 成员运算，判断一个元素是否不在一个序列中 | 4 not in [1, 2, 3, 4, 5] |

## 身份运算

身份运算符用于比较对象值的内存地址是否相等，如果是相同的内存地址，说明它们实际上是同一个对象，返回 True，否则返回 False。

| 运算符 | 简单描述                                 | 例子       |
| ------ | ---------------------------------------- | ---------- |
| is     | 身份运算，判断两个对象是否引用同一个对象 | a is b     |
| is not | 身份运算，判断两个对象是否引用不同的对象 | a is not b |

## 位运算

| 运算符 | 简单描述                               | 例子        |
| ------ | -------------------------------------- | ----------- |
| &      | 位与，将两个数的二进制位进行与运算     | 4 & 5 = 4   |
| \|     | 位或，将两个数的二进制位进行或运算     | 4 \| 5 = 5  |
| ^      | 位异或，将两个数的二进制位进行异或运算 | 4 ^ 5 = 1   |
| <<     | 左移，将一个数的二进制位向左移动       | 4 << 2 = 16 |
| >>     | 右移，将一个数的二进制位向右移动       | 4 >> 2 = 1  |
| ~      | 位取反，将一个数的二进制位取反         | ~4 = -5     |

## 赋值运算和复合赋值

- `=` 赋值运算符用于将一个值赋值给一个变量，赋值运算符的左边是变量，右边是值。
- 复合赋值运算符用于将一个常规运算符和赋值运算符组合起来，简化代码书写。当 python 解释器执行到复合赋值运算符时，先算复合赋值运算符右侧的表达式，然后再算复合赋值运算符的算数运算，最后算赋值运算。

比如 `a += 1 + 2` ，先计算右侧表达式 `1 + 2`，然后再计算 `a + 3`，最后将计算结果赋值给 `a`。

| 运算符 | 简单描述                                 | 例子      |
| ------ | ---------------------------------------- | --------- |
| =      | 赋值运算符，将一个值赋值给一个变量       | a = 1     |
| +=     | 复合赋值运算符，将一个值添加到变量上     | a += 1    |
| -=     | 复合赋值运算符，将一个值从变量上减去     | a -= 1    |
| \*=    | 复合赋值运算符，将一个值乘到变量上       | a \*= 2   |
| /=     | 复合赋值运算符，将一个值除到变量上       | a /= 2    |
| //=    | 复合赋值运算符，将一个值整除到变量上     | a //= 2   |
| %=     | 复合赋值运算符，将一个值取模到变量上     | a %= 2    |
| \*\*=  | 复合赋值运算符，将一个值乘方到变量上     | a \*\*= 2 |
| <<=    | 复合赋值运算符，将一个值左移到变量上     | a <<= 2   |
| >>=    | 复合赋值运算符，将一个值右移到变量上     | a >>= 2   |
| &=     | 复合赋值运算符，将一个值按位与到变量上   | a &= 2    |
| \|=    | 复合赋值运算符，将一个值按位或到变量上   | a \|= 2   |
| ^=     | 复合赋值运算符，将一个值按位异或到变量上 | a ^= 2    |

## 海象运算符 walrus operator

海象运算符（walrus operator）是Python 3.8新引入的，因为看着像一张有长牙的笑脸（像海象）。通常用于 `if / for / while` 等表达式中直接定义局部引用变量，优化性能。

```python
# Without walrus operator
n = len("hello world")
if n > 5:
   print(f"Length is {n}")

# With walrus operator
if (n := len("hello world")) > 5:
   print(f"Length is {n}") # Output: Length is 11
else:
  print(f"Length is {n}") # 不满足条件时，在 else 或 elif 中都可以访问

# 这里避免了在多处重复调用 input 函数
line = input("Enter: ")
while line != "quit":
   print(line)
   line = input("Enter: ")
# With walrus operator
while (line := input("Enter: ")) != "quit":
   print(line)

# 这通过存储中间结果 l，避免每次迭代都调用 len 函数，提高了效率。
nums = ["22", "hello", "world", "Python"]
# Without walrus: len(x) called twice
result = [x for x in nums if len(x) > 3]
# With walrus: len(x) called once
result = [x for x in nums if (l := len(x)) > 3]
print(result) # ['hello', 'world', 'Python']
```

## 优先级

上述运算符如果组合使用，需要根据优先级进行计算。优先级从高到低依次为：

1. 括号
2. 位运算符
3. 算术运算符
4. 关系运算符
5. 逻辑运算符
6. 身份运算符
7. 成员运算符
8. 赋值运算符

在使用时，需要注意运算符的优先级，避免出现错误的计算结果。
