## 编程语言的分类

从不同的维度来区分各种编程语言的不同，通常从以下几个维度进行分类：

### 按抽象层次划分：低级语言和高级语言

计算机 CPU 执行二进制指令，也称为机器语言，所有语言编写的程序最终都要翻译成二进制的指令代码才能被计算机执行。

越低级的语言，形式上越接近机器指令，其中汇编语言就是与机器指令一一对应的低级语言。高级语言对底层操作进行了抽象和封装，越高级的语言，一条语句对应的指令数可能很多，这样使编写程序的过程更符合人类的思维习惯。也就是说你用高级语言写一句，会被转换成许多底层操作，大部分的工作交给了负责转换的编译器，从而人力得到了解放。因为机器就是用来为人类提供便利的，所以说高级语言的出现是计算机发展的必然结果。

- 低级语言：直接操作硬件的语言，如 C、C++ 等。
- 高级语言：提供了更高层次的抽象，如 Python、Java、C# 等。

### 按是否需要编译划分：编译型语言和解释型语言

- 编译型语言：程序执行前，需要使用编译器将源代码转换为机器语言的可执行文件，然后计算机执行可执行文件，从而运行程序。比如 C、C++
- 解释型语言：编写的程序可以直接在计算机上运行，不需要编译，解释器边解释源代码，边执行。比如 Python、JavaScript、PHP、Ruby 等。
- 混合型语言（编译解释都有）：程序执行前，需要先使用编译器将源代码转换为中间代码，再使用解释器将中间代码转换为机器语言。比如 Java 会先编译成字节码，再由 JVM 解释器执行。

严格区分一门语言是编译型还是解释型语言已经不太准确，很多语言在发展过程中，为了对性能的改善，会逐渐优化。比如 JavaScript 的 JIT 技术，它会在运行时会将一些需要频繁执行的热点代码转换为机器语言，从而提高执行效率。

### 按类型系统划分：强弱类型和动静类型

- 强类型语言：在运行时不允许隐式类型转换，初始化时固定了类型，之后就不能改变该值的类型了。比如 C、C++、Java、C#、Python、 Go 等。
- 弱类型语言：在运行时允许隐式类型转换。字符串和数字相加，会尝试先将字符串转换数字，然后执行相加操作。比如 JavaScript 等。
- 静态类型语言：在编程时需要严格定义变量的类型，在编译时进行类型检查，确保所有变量都有明确的类型。比如 C、C++、Java、C# 等。
- 动态类型语言：变量定义没有类型要求，变量的类型可以根据上下文变化。比如 Python、JavaScript、PHP、Ruby 等。

上述特殊的属于 Python，既是强类型语言，也是动态类型语言。

另外对于静态类型和动态类型语言的区分，也随着技术不断发展的，语言也在不断的优化。比如 Python 3 引入了类型注解，JavaScript 语言中也出现了需要类型声明的 TypeScript。

### 按执行方式划分：脚本语言和编译型语言

- 脚本语言：不需要编译，也不需要构建，文件可以直接执行。比如 Python、JavaScript 等。
- 编译型语言：需要编译成一个可执行文件才能被计算机执行。比如 C 必须构建成一个 Jar 包、C++ 必须构建成一个可执行文件 .exe。

## Python 历史

- 起源：由荷兰程序员吉多・范罗苏姆（Guido van Rossum） 于 1989 年圣诞节期间构思，旨在设计一种语法简洁、可读性强的语言，反对当时其他语言中复杂繁琐的语法结构。
- 诞生：1991 年，Python 第一个公开版本发布，核心设计理念是 “优雅、明确、简单”，强调代码的可读性和开发者效率，采用缩进来划分代码块，而非传统的大括号。
- 发展：
  - 2000 年，Python 2.0 发布，引入列表推导式、垃圾回收等重要特性。
  - 2008 年，Python 3.0 发布，这是一个不兼容旧版本的重大破坏性更新，优化了语法一致性和功能（如统一字符串类型、改进整数除法等）。
  - 2020 年，Python 2 停止维护，Python 3 成为目前的主流版本。
- 现状：凭借广泛的适用性（web 开发、数据分析、人工智能等领域）和活跃的社区支持，Python 已成为全球最流行的编程语言之一。

## Python logo 蟒蛇

Python Logo 的设计灵感来源于其创始人吉多·范罗苏姆（Guido van Rossum）的喜好。吉多是一位喜剧迷，尤其钟爱英国喜剧团体 Monty Python。这个团体以其独特的幽默风格和讽刺精神在 1970 年代广受欢迎。因此，当吉多在 1989 年圣诞节期间决定开发一种新的脚本解释程序时，他选择了“Python”这个名字，以此来纪念他喜爱的喜剧团体。

## Python 应用领域

- Web 开发：Python 有丰富的 Web 框架，如 Django、Flask、FastAPI 等，用于构建动态网站和 Web 应用程序。
- 数据分析：Python 有强大的数据处理库，如 NumPy、Pandas、Matplotlib、Seaborn 等，用于数据分析和可视化。
- 人工智能：Python 有丰富的 AI 库，如 TensorFlow、PyTorch 等，用于机器学习和深度学习。
- 游戏开发：Python 有游戏开发库，如 Pygame、PyOpenGL 等，用于创建 2D 和 3D 游戏。

## Python 解释器

Python 解释器是 Python 代码的运行环境，负责将 Python 代码翻译成计算机可以执行的指令。Python 解释器有多种实现，包括 CPython、Jython、IronPython 等。

- CPython：是 Python 官方解释器的实现，是最常用的 Python 解释器。它是用 C 语言编写的，因此执行速度快。
- Jython：是 Python 解释器的一种实现，它将 Python 代码翻译成 Java 字节码，然后在 JVM 上运行。这使得 Python 代码可以在 Java 平台上运行。
- IronPython：是 Python 解释器的一种实现，它将 Python 代码翻译成 .NET 字节码，然后在 CLR 上运行。这使得 Python 代码可以在 .NET 平台上运行。
- PyPy：是 Python 解释器的另一种实现，它使用 Just-In-Time 编译（JIT）技术，将热点代码翻译成机器语言，从而提高执行速度。
