# Python 安装和使用

## 安装

直接到 [Python 官网](https://www.python.org/) 下载安装包，下一步下一步安装即可。

安装成功后，在终端中输入 `python --version` 查看版本号。如果显示正确，即安装成功。

## 运行

安装好 Python 之后，那么如何运行 Python 程序呢？一般有两种方式。

### 交互式解释器

在终端中输入 `python` 即可打开交互式解释器。然后在交互式解释器中输入代码，然后回车即可运行。

### python IDLE

IDLE 是 Python 安装时自带的一个简单的编辑器。可以在 window 菜单中找到 IDLE 选项，并打开。然后在 IDLE 中输入代码，然后点击运行按钮即可运行。

### 专业的代码编辑器

- vscode: 这是一个免费的、开源的、跨平台的代码编辑器。可以在 [vscode 官网](https://code.visualstudio.com/) 下载安装包，下一步下一步安装即可。
- pycharm: 这是一个商业的代码编辑器。但是初学者可以使用社区版。可以在 [pycharm 官网](https://www.jetbrains.com/pycharm/) 下载安装包，下一步下一步安装即可。
