# File 文件操作

Python的文件操作主要围绕内置的 `open()` 函数

## 前置知识

1. 文件路径 path：定位文件的位置。

- 相对路径：相对于当前程序运行目录的路径，例如 `data.txt` 或 `./folder/data.txt`。
- 绝对路径：从根目录开始的完整路径，例如 `C:/Users/test/data.txt` (Windows) 或 `/home/user/data.txt` (Linux/Mac)。
- 数据持久化：将程序运行时的内存数据保存到硬盘文件中，这样即使程序关闭，数据也不会丢失。文件操作是实现数据持久化的基础。
- 文本与二进制：
  - 文本文件：存储人类可读的字符，操作时需要指定编码（如 utf-8）。
  - 二进制文件：存储图片、视频、音频等非文本数据，操作时以字节为单位，无需指定编码。

## 打开文件

`open()` 函数，其基本语法为 `open(文件路径, 模式, 编码)`

- 文件路径：通常使用 pathlib 模块的 Path 类来表示文件路径，如 `Path('data.txt')`。
- 模式：指定文件打开的方式，如 `'r'` 表示只读，`'w'` 表示只写，`'a'` 表示追加，`'x'` 表示创建，`'b'` 表示二进制模式，`'t'` 表示文本模式。
- 编码：指定文件编码，默认 `'utf-8'`。

| 模式   | 说明                                                | 适用场景             |
| :----- | :-------------------------------------------------- | :------------------- |
| `'r'`  | 只读（默认模式），文件不存在会报错                  | 读取现有文件内容     |
| `'w'`  | 只写，会覆盖原文件内容，文件不存在则创建            | 创建新文件或完全重写 |
| `'a'`  | 追加写，在文件末尾添加内容，不覆盖                  | 日志记录、添加数据   |
| `'r+'` | 读写模式，文件指针在开头                            | 需要先读后写的场景   |
| `'b'`  | 二进制模式，与其他模式组合使用（如 `'rb'`, `'wb'`） | 处理图片、视频等     |
| `'x'`  | 独占创建，仅在文件不存在时创建，否则报错            | 安全地创建新文件     |

## 读取文件

- 读取全部内容：`file.read()` 返回一个包含文件所有内容的字符串。适合小文件。
- 读取所有行：`file.readlines()` 返回一个列表，每个元素是文件的一行。
- 逐行读取（推荐）：直接迭代文件对象 `for line in file:`。这种方式内存效率最高，尤其适合处理大文件，因为它不会一次性将所有内容加载到内存中。

## 写入文件

- 写入字符串：`file.write`("要写入的内容")。
- 写入多行：`file.writelines([列表])`，将一个字符串列表写入文件。注意，此方法不会自动添加换行符，需要自行在字符串中包含 `\n`。

## 代码示例

最佳实践是使用 `with` 语句来操作文件，因为它会在代码块执行完毕后自动关闭文件，即使发生异常也能确保资源被正确释放，避免了手动调用 `close()` 可能导致的资源泄露问题。

```python
# --- 读取文件 ---
# 方式一：读取全部内容
with open("example.txt", "r", encoding="utf-8") as f:
    content = f.read()
    print(content)

# 方式二：逐行读取（内存最优）
with open("example.txt", "r", encoding="utf-8") as f:
    for line in f:
        print(line.strip()) # strip() 用于去除行尾的换行符

# --- 写入文件 ---
# 覆盖写入
with open("output.txt", "w", encoding="utf-8") as f:
    f.write("这是第一行\n")
    f.write("这是第二行\n")

# 追加写入
with open("output.txt", "a", encoding="utf-8") as f:
    f.write("这是追加的内容\n")

# --- 二进制文件操作 (例如复制图片) ---
with open("source.jpg", "rb") as f_src:
    data = f_src.read()
with open("target.jpg", "wb") as f_dst:
    f_dst.write(data)
```

配合 pathlib 可以更方便的操作文件

```python
from pathlib import Path

# 创建Path对象
p = Path("my_folder") / "my_file.txt" # 使用 '/' 操作符拼接路径

# 检查路径是否存在
if p.exists():
    print("文件存在")

# 检查是否是文件
if p.is_file():
    print("这是一个文件")

# 获取文件名和父目录
print(p.name)      # 输出: my_file.txt
print(p.parent)    # 输出: my_folder
```

## 注意事项

1. 忘记关闭文件：始终使用 `with` 语句来自动管理文件关闭，这是最重要的最佳实践。
1. 错误的写入模式：使用 `'w'` 模式会清空并覆盖整个文件内容。如果只是想添加内容，务必使用 `'a'` 模式。
1. 编码问题：在读写文本文件时，显式指定 `encoding="utf-8"` 可以避免在不同操作系统间出现乱码问题。
1. 大文件读取：避免对大文件使用 `f.read()`，因为它会一次性将所有内容加载到内存，可能导致内存溢出。应使用 `for line in f:` 逐行处理。
1. 路径兼容性：使用 `pathlib` 或 `os.path.join()` 来处理路径拼接，而不是手动使用字符串拼接，以确保代码在不同操作系统（Windows/Linux/Mac）上的兼容性。
