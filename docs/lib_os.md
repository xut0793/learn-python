# OS 系统模块

os 模块是 Python 标准库中与操作系统交互的基石，它提供了一套跨平台的接口，让你无需关心底层是 Windows、Linux 还是 macOS，就能轻松管理文件、目录和系统环境。

主要操作方法分类：

1. 进程与环境：管理环境变量、执行系统命令、获取进程信息。
2. 路径处理：通过 os.path 子模块处理路径字符串（拼接、分割、规范化），在现代 python，大部分操作建议通过 pathlib 模块完成。
3. 文件系统交互：文件与目录的增删改查、遍历。
4. 系统感知：获取操作系统类型、用户信息、硬件资源等。

进程与环境操作：

| 方法名               | 功能描述                     | 专家提示                                             |
| :------------------- | :--------------------------- | :--------------------------------------------------- |
| `os.name`            | 返回操作系统类型字符串。     | Windows 返回 `'nt'`，Linux/Mac 返回 `'posix'`。      |
| `os.environ`         | 获取系统环境变量的字典对象。 | 使用 `os.getenv('KEY')` 获取更安全。                 |
| `os.system(command)` | 在系统 Shell 中执行命令。    | 不推荐用于复杂任务，建议使用 `subprocess` 模块替代。 |
| `os.getpid()`        | 获取当前进程的 ID。          | 常用于日志记录或进程管理。                           |
| `os.sep`             | 当前操作系统的路径分隔符。   | Windows 是 `\`，Linux 是 `/`。                       |

路径处理

| 方法名                       | 功能描述                           | 专家提示                                       |
| :--------------------------- | :--------------------------------- | :--------------------------------------------- |
| `os.path.join(path, *paths)` | 智能拼接路径。                     | 自动处理 `/` 和 `\` 的分隔符差异，最常用。     |
| `os.path.exists(path)`       | 判断路径（文件或目录）是否存在。   | 操作文件前建议先检查，防止报错。               |
| `os.path.isfile(path)`       | 判断路径是否是一个文件。           | 区分文件与目录的关键。                         |
| `os.path.isdir(path)`        | 判断路径是否是一个目录。           | 常用于遍历目录时的逻辑判断。                   |
| `os.path.split(path)`        | 分割路径为 `(目录, 文件名)`。      | 例如 `('/a/b', 'c.txt')`。                     |
| `os.path.splitext(path)`     | 分割文件名与扩展名。               | 例如 `('c', '.txt')`，常用于筛选特定类型文件。 |
| `os.path.abspath(path)`      | 获取绝对路径。                     | 将相对路径转换为完整路径。                     |
| `os.path.basename(path)`     | 获取路径中的最后一部分（文件名）。 | `os.path.split` 的快捷方式。                   |

文件操作
| 方法名 | 功能描述 | 专家提示 |
| :--- | :--- | :--- |
| `os.remove(path)` | 删除指定文件。 | 别名 `os.unlink(path)`，无法删除目录。 |
| `os.rename(src, dst)` | 重命名文件或移动文件位置。 | 如果目标已存在，在 Windows 上可能会报错。 |
| `os.replace(src, dst)` | 强制重命名或移动。 | 如果目标存在，会无条件覆盖（原子操作）。 |
| `os.stat(path)` | 获取文件/目录的状态信息。 | 返回对象包含文件大小 (`st_size`)、修改时间 (`st_mtime`) 等。 |
| `os.path.getsize(path)` | 获取文件大小（字节）。 | 实际上是 `os.stat` 的便捷封装。 |
| `os.path.getmtime(path)` | 获取文件最后修改时间。 | 返回时间戳，通常需配合 `time` 模块格式化。 |
